## [Unreleased]

### Added

## [1.0.0] - 2026-07-22

### Added

- First official release of the CeloHT flagship repository.
- Full documentation set: vision, mission, values, architecture,
  governance, security, and more under `docs/`.
- Standard GitHub community files: README, LICENSE, CODE_OF_CONDUCT,
  CONTRIBUTING, SECURITY, SUPPORT, GOVERNANCE, ROADMAP, AUTHORS, NOTICE,
  CODEOWNERS, CITATION.cff.
- GitHub configuration: Issue/Discussion/PR templates, Dependabot, GitHub Pages.
- GitHub Actions workflows: CI, Markdown Lint, Link Checker, CodeQL, Secret
  Scanning, Dependency Review, Release Automation, Pages Deployment.
- Official branding assets (logo) under `assets/branding/`.
- An introductory example under `examples/hello-agent/`.
