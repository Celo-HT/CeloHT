# Agent Network (Pillar 2)

## The Goal

Build and support a decentralized network of community agents who handle
cash-in, cash-out, remittances, peer-to-peer payments, and local financial
access through the Celo ecosystem especially in areas where traditional
banking hasn't reached.

## What a Community Agent Does

A CeloHT agent is someone from the local community who:

1. Has completed CeloHT's education program (see [`education.md`](education.md))
2. Facilitates cash-to-cUSD exchanges for people in the community
3. Helps new users set up and understand Valora
4. Acts as a local point of contact for basic questions and support

## How Agents Operate

- **Fee transparency**: agents must clearly communicate any fee before a
  transaction happens.
- **Basic identity verification**: a lightweight process adapted to local
  context, designed to reduce fraud without creating unreasonable barriers.
- **Ongoing training**: agents attend refresher sessions on a regular basis.
- **Code of conduct**: agents follow specific standards designed to protect
  users from exploitation.

## Becoming an Agent

1. Complete the full education program
2. Demonstrate a solid understanding of the basics Valora, seed phrase
   security
3. Commit to the community's operational standards
4. Get sign-off from a local network coordinator

## Connection to Reforestation

Some community agents also take on responsibility for coordinating local
reforestation activity, creating a direct link between financial inclusion
and environmental sustainability see [`reforestation.md`](reforestation.md).
