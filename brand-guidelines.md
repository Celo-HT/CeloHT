# Brand Guidelines

## Visual Identity

### Official Colors

| Color | Hex | Use |
|---|---|---|
| Navy (primary) | `#0B1120` | Background, primary text, authority |
| Gold (accent) | `#F5C842` | Accents, calls to action, highlighted elements |

### Typography

| Purpose | Typeface | Use |
|---|---|---|
| Headings | Cormorant Garamond | Primary headings, editorial elements |
| Body | Montserrat | Body text, interface, documentation |

## Brand Voice

- **Clear**: avoid technical jargon without explanation
- **Respectful**: never pushy; always educational first
- **Proud but humble**: proud of the mission, without exaggeration or
  irresponsible promises
- **Plain English first**: our primary language for documentation and
  international communication, with Haitian Creole as our founding language
  for community-facing material

## How to Describe CeloHT

**Always say**: "A community-driven, open-source initiative built on the
Celo ecosystem."

**Never say**: "a blockchain," "a cryptocurrency," "an investment," "a DeFi
platform," and so on see [`legal-status.md`](legal-status.md) for the
complete list.

## Consistent Application

The three pillars (Education, Agent Network, Reforestation) should appear
consistently across brand materials, documents, and public communication to
reinforce a unified project identity.

See [`logo-usage.md`](logo-usage.md) for specific logo usage rules, and
[`media-kit.md`](media-kit.md) for downloadable assets.
