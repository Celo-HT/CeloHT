# Contributing (Documentation Summary)

For the full contribution guide, see [`CONTRIBUTING.md`](../CONTRIBUTING.md)
at the root of the repository — it covers the Pull Request process, quality
standards, and commit message format.

## Quick Ways to Contribute

- **Improve documentation**: fix errors, clarify explanations, add examples
- **Translation**: help translate material between Haitian Creole, English,
  and (eventually) Spanish
- **Examples**: add new examples under `examples/`
- **Tooling**: improve validation scripts under `scripts/` and `tools/`

## Before You Start

1. Read [`CODE_OF_CONDUCT.md`](../CODE_OF_CONDUCT.md)
2. Check [`repository-guide.md`](repository-guide.md) to understand the layout
3. Search [Issues](https://github.com/celo-ht/celoht/issues) to see if
   someone's already working on the same thing
