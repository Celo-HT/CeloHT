# Governance

This document summarizes governance in the context of the broader
documentation set. For the full operational detail — structure, decision
process, and how disputes get resolved — see [`GOVERNANCE.md`](../GOVERNANCE.md)
at the root of the repository.

## Summary

CeloHT is run by a **Foundation Director**, a **Maintainer Council**, and
open **community** contribution. Minor changes get approved by individual
maintainers; major changes require Council consensus and public discussion.

## Core Principles

1. Transparency: every significant decision is documented publicly.
2. Accountability: each role has clearly defined responsibility (see
   [`maintainers.md`](maintainers.md)).
3. Evolution: this model is meant to grow toward more decentralization over
   time.

See [`GOVERNANCE.md`](../GOVERNANCE.md) for the full policy text.
