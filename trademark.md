## Name and Logo

"CeloHT" and its official logo (available under
[`assets/branding/`](../assets/branding/)) are community identifiers for
this project. They are not Celo Foundation trademarks, and their use does
not imply any official affiliation with the Celo Foundation or with Valora.

## Uses That Don't Require Specific Permission

- Referencing CeloHT by name in articles, reports, or educational content
- Linking to the official repository or website
- Using the logo **unmodified** to identify CeloHT specifically as a content
  source (for example: "This material comes from CeloHT")

## Uses That Require Written Permission

- Modifying the logo (color, shape, composition) — see
  [`logo-usage.md`](logo-usage.md) for specific rules
- Using the CeloHT name or logo in a product, service, or domain name
- Using the mark in a way that suggests an endorsement or partnership that
  doesn't exist

## Absolutely Prohibited

- Using the CeloHT name or logo to promote a different cryptocurrency,
  token, or ICO
- Using the mark in content that misrepresents what CeloHT is (see
  [`legal-status.md`](legal-status.md) for what CeloHT is not)
